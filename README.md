# ⛓️ Silverway — Decentralized Exchange on LitVM

**Silverway** is a seamless decentralized exchange (DEX) and automated market maker (AMM) built for trading precision on the **Litecoin Virtual Machine (LitVM)**. It provides a fluid, minimalist interface for users to swap Litecoin-native assets, manage liquidity, and explore the LitVM ecosystem.

🔗 **Live Application:** <https://silverway-lit-vm.vercel.app/>

## ✨ Features

* **💱 Precision Swaps:** Trade `zkLTC`, `USDC`, and other L2 assets with minimal slippage.

* **🔌 One-Click Network Integration:** Automatically prompt users to add the LitVM LiteForge network to their MetaMask wallet directly from the navbar.

* **💧 Faucet Access:** Quick links to the Caldera faucet to easily fund testnet wallets.

* **🎨 Minimalist "Carbon Black" UI:** Built with Tailwind CSS for a premium, distraction-free trading experience.

* **⚡ Fast & Responsive:** Powered by React and Vite for lightning-fast load times.

## 🌐 Network Details (LitVM LiteForge)

Silverway is currently configured for the LitVM LiteForge Testnet:

* **Network Name:** LitVM LiteForge

* **RPC URL:** `https://liteforge.rpc.caldera.xyz/http`

* **Chain ID:** `4441` (`0x1159` in hex)

* **Currency Symbol:** `zkLTC`

* **Block Explorer:** <https://liteforge.explorer.caldera.xyz>

## 🛠️ Tech Stack

* **Frontend Framework:** React 18 (Vite)

* **Styling:** Tailwind CSS + custom Apple-style minimal design tokens

* **Icons:** Lucide React

* **UI Components:** Radix UI / shadcn

* **Web3 Interaction:** Ethers.js / window.ethereum API

* **Deployment:** Vercel
